## 문제
$- 1 \leq t \leq 1$인 실수 $t$에 대하여 $x$에 대한 방정식

$(\sin\,\frac{\pi\, x}{2} - t)(\cos\,\frac{\pi\, x}{2} - t) $ $= 0$

의 실근 중에서 집합 $\{ x   \mid    0 \leq x< 4\}$에 속하는 가장 작은 값을 $\alpha(t)$, 가장 큰 값을 $\beta(t)$라 하자. \<보기\>에서 옳은 것만을 있는 대로 고른 것은? \[4점\]

|                                                      | \| 보기 \|                              |                                                      |
+------------------------------------------------------+                                         +------------------------------------------------------+
|                                                      |                                         |                                                      |
+------------+-----------------------------------------+-----------------------------------------+-----------------------------------------+------------+
|            | ㄱ. $- 1 \leq t< 0$인 모든 실수 $t$에 대하여 $\alpha(t) + \beta(t) $ $= 5$이다.                                         |            |
|            |                                                                                                                             |            |
|            | ㄴ. $\{ t\,\, \mid \,\,\beta(t) - \alpha(t) = \beta(0) - \alpha(0)\} $ $= \{ t |  0 \leq t \leq \frac{\sqrt{2}}{2}\}$       |            |
|            |                                                                                                                             |            |
|            | ㄷ. $\alpha(t_{1}) $ $= \alpha(t_{2})$인 두 실수 $t_{1},\ t_{2}$에 대하여\                                                    |            |

## 풀이
ㄱ. 방정식\
$(\sin\frac{\pi x}{2} - t)(\cos\frac{\pi x}{2} - t) = 0$ 에서\
$\sin\frac{\pi x}{2} = t$ 또는 $\cos\frac{\pi x}{2} = t$
이 방정식의 실근은 두 함수\
$y = \sin\frac{\pi x}{2},$ $y = \cos\frac{\pi x}{2}$의 그래프와 $y = t$와의 교점의 $x$좌표이다.\
한편, 두 함수 $y = \sin\frac{\pi x}{2},$ $y = \cos\frac{\pi x}{2}$의 주기가 모두 $4$이므로 다음과 같다.\
![](/images/docx2mdx/220615-20260429-123318/media/image1.webp)\
$- 1 \leq t< 0$이면 직선 $y = t$와 $\alpha(t),$ $\beta(t)$는 다음 그림과 같다.\
![](/images/docx2mdx/220615-20260429-123318/media/image2.webp)\
이때, 함수 $y = \cos\frac{\pi x}{2}$의 그래프는 함수 $y = \sin\frac{\pi x}{2}$의 그래프를 평행이동시키면 겹쳐질 수 있고 함수 $y = \sin\frac{\pi x}{2}$의 그래프는 직선 $x = 1,$ $x = 3$에 대하여 대칭이고 점 $(2,\ 0)$에 대하여 대칭이다.\
그러므로\
$\alpha(t) = 1 + k$ $(0< k \leq 1)$
로 놓으면\
$\beta(t) = 4 - k$
그러므로\
$\alpha(t) + \beta(t) = 5$ \<참\>

ㄴ. 실근 $\alpha(t),$ $\beta(t)$는 집합 $\{ x   \mid    0 \leq x< 4\}$의 원소이므로\
$\beta(0) = 3,$ $\alpha(0) = 0$
그러므로 주어진 식은

$\{ t\,\, \mid \,\,\beta(t) - \alpha(t) = \beta(0) - \alpha(0)\} $ $= \{ t   \mid   \beta(t) - \alpha(t) = 3\}$

(ⅰ) $0 \leq t \leq \frac{\sqrt{2}}{2}$일 때,\
$t = 0$이면 $\beta(0) - \alpha(0) = 3 - 0 = 3$
$t \neq 0$이면 다음 그림과 같다.\
![](/images/docx2mdx/220615-20260429-123318/media/image3.webp)\
이때, $\alpha(t) = k$ $(0< k \leq \frac{1}{2})$\
이라 하면\
$\beta(t) = 3 + k$
그러므로 $\beta(t) - \alpha(t) = 3$

(ⅱ) $\frac{\sqrt{2}}{2}< t< 1$일 때,\
![](/images/docx2mdx/220615-20260429-123318/media/image4.webp)\
이때, $\alpha(t) = k$ $(0< k<\frac{1}{2})$\
이라 하면\
$\beta(t) = 4 - k$
그러므로\
$\beta(t) - \alpha(t) = 4 - 2k$ $(0< 2k< 1)$

(ⅲ) $t = 1$일 때,\
$\alpha(1) = 0,$ $\beta(1) = 1$이므로\
$\beta(1) - \alpha(1) = 1$

(ⅳ) $- 1 \leq t< 0$일 때,\
![](/images/docx2mdx/220615-20260429-123318/media/image5.webp)\
$1<\alpha(t) \leq 2,$ $3 \leq \beta(t)< 4$이므로\
$\beta(t) - \alpha(t)< 3$

따라서 (ⅰ), (ⅱ), (ⅲ), (ⅳ)에서\
$\{ t   \mid   \beta(t) - \alpha(t) = 3\} $ $= \{ t|0 \leq t \leq \frac{\sqrt{2}}{2}\}$ \<참\>

ㄷ. $\alpha(t_{1}) = \alpha(t_{2})$이기 위해서는\
$0< t_{1}<\frac{\sqrt{2}}{2}< t_{2}$
이때, $\alpha(t_{1}) = \alpha(t_{2}) = \alpha$라 하면\
$t_{1} = \sin\frac{\pi}{2}\alpha,$ $t_{2} = \cos\frac{\pi}{2}\alpha$
이때, $t_{2} = t_{1} + \frac{1}{2}$이므로\
$\cos\frac{\pi}{2}\alpha = \sin\frac{\pi}{2}\alpha + \frac{1}{2}$
이 식을 $\cos^{2}\frac{\pi}{2}\alpha + \sin^{2}\frac{\pi}{2}\alpha = 1$에 대입하면\
$2\sin^{2}\frac{\pi}{2}\alpha + \sin\frac{\pi}{2}\alpha + \frac{1}{4} = 1$
$8\sin^{2}\frac{\pi}{2}\alpha + 4\sin\frac{\pi}{2}\alpha - 3 = 0$
$\sin\frac{\pi}{2}\alpha = \frac{- 2 \pm \sqrt{28}}{8}$
이때, $\sin\frac{\pi}{2}\alpha> 0$이므로\
$\sin\frac{\pi}{2}\alpha = \frac{- 1 + \sqrt{7}}{4}$
그러므로\
$t_{1} = \frac{- 1 + \sqrt{7}}{4},$
$t_{2} = t_{1} + \frac{1}{2} = \frac{1 + \sqrt{7}}{4}$
따라서\
$t_{1} \times t_{2} = \frac{( - 1 + \sqrt{7})(1 + \sqrt{7})}{16} = \frac{3}{8}$ \<거짓\>
