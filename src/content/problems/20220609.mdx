## 문제
수열 $\{ a_{n}\}$이 모든 자연수 $n$에 대하여

$$
a_{n+1} = 
\begin{cases}
\dfrac{1}{a_n} & (n \text{이 홀수인 경우}) \\
8a_n & (n \text{이 짝수인 경우})
\end{cases}
$$

이고 $a_{12} $ $= \frac{1}{2}$일 때, $a_{1} + a_{4}$의 값은? \[4점\]

① $\frac{3}{4}$ ② $\frac{9}{4}$ ③ $\frac{5}{2}$ ④ $\frac{17}{4}$ ⑤ $\frac{9}{2}$


## 힌트
$a_{n+1}$은 $n$이 홀수일 때와 짝수일 때 식이 달라요. $a_{12}$에서 **한 항 앞**인 $a_{11}$을 구하려면, 관계식에서 $n = 11$인 경우를 보면 돼요. $11$이 홀수인지 짝수인지부터 확인해 보세요.

## 힌트
$n$이 홀수이면 $a_{n+1} = \dfrac{1}{a_n}$이니까, 이미 알고 있는 $a_{n+1}$로부터 $a_n$은 **역수**로 구할 수 있어요. $n$이 짝수이면 $a_{n+1} = 8a_n$이니까 $a_n$은 $a_{n+1}$을 **$8$로 나눈 값**이에요.

## 힌트
$a_{12}$에서 출발해서 $a_{11}$, $a_{10}$, … 처럼 **항 번호를 하나씩 줄여 가며** 같은 방법을 반복하면 $a_1$과 $a_4$까지 갈 수 있어요. 매번 그때의 $n$이 홀수인지 짝수인지만 헷갈리지 않게 표시해 두면 좋아요.

## 힌트
항들이 $\frac{1}{2}$, $2$, $\frac{1}{4}$, $4$ 같은 값으로만 돌아가는 패턴이 보이면, 계산 실수를 줄일 수 있어요.

## 풀이
주어진 관계는 자연수 $n$에 대하여

- $n$이 **홀수**이면 $a_{n+1} = \dfrac{1}{a_n}$

- $n$이 **짝수**이면 $a_{n+1} = 8a_n$

이에요.

**앞항을 뒤에서 구하기**로 정리하면, $a_{n+1}$이 알려져 있을 때

- $n$이 홀수이면 $a_n = \dfrac{1}{a_{n+1}}$

- $n$이 짝수이면 $a_n = \dfrac{a_{n+1}}{8}$

이에요.

문제에서 $a_{12} = \dfrac{1}{2}$이에요.

$n = 11$은 홀수이므로 $a_{11} = \dfrac{1}{a_{12}} = \dfrac{1}{\frac{1}{2}} = 2$이에요.

$n = 10$은 짝수이므로 $a_{10} = \dfrac{a_{11}}{8} = \dfrac{2}{8} = \dfrac{1}{4}$이에요.

$n = 9$는 홀수이므로 $a_9 = \dfrac{1}{a_{10}} = \dfrac{1}{\frac{1}{4}} = 4$이에요.

$n = 8$은 짝수이므로 $a_8 = \dfrac{a_9}{8} = \dfrac{4}{8} = \dfrac{1}{2}$이에요.

$n = 7$은 홀수이므로 $a_7 = \dfrac{1}{a_8} = \dfrac{1}{\frac{1}{2}} = 2$이에요.

$n = 6$은 짝수이므로 $a_6 = \dfrac{a_7}{8} = \dfrac{2}{8} = \dfrac{1}{4}$이에요.

$n = 5$는 홀수이므로 $a_5 = \dfrac{1}{a_6} = \dfrac{1}{\frac{1}{4}} = 4$이에요.

$n = 4$는 짝수이므로 $a_4 = \dfrac{a_5}{8} = \dfrac{4}{8} = \dfrac{1}{2}$이에요.

$n = 3$은 홀수이므로 $a_3 = \dfrac{1}{a_4} = \dfrac{1}{\frac{1}{2}} = 2$이에요.

$n = 2$는 짝수이므로 $a_2 = \dfrac{a_3}{8} = \dfrac{2}{8} = \dfrac{1}{4}$이에요.

$n = 1$은 홀수이므로 $a_1 = \dfrac{1}{a_2} = \dfrac{1}{\frac{1}{4}} = 4$이에요.

따라서

$a_1 + a_4 = 4 + \dfrac{1}{2} = \dfrac{9}{2}$

이에요.

정답은 **⑤**이에요.
