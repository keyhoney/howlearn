## 문제
함수 $f(x) $ $= x^{3} - 3x + 12$가 $x = a$에서 극소일 때, $a + f(a)$의 값을 구하시오. (단, $a$는 상수이다.) \[3점\]

## 풀이
$f '(x) = 3x^{2} - 3 = 3(x + 1)(x - 1)$

$f '(x) = 0$에서

$x = - 1$ 또는 $x = 1$

함수 $f(x)$의 증가와 감소를 표로 나타내면 다음과 같다.

  ----------- ---------- ------- ---------- ------- ----------
      $x$      $\cdots$   $- 1$   $\cdots$    $1$    $\cdots$

   $f '(x)$     $+$       $0$      $-$       $0$      $+$

    $f(x)$        ↗       극대       ↘       극소       ↗
  ----------- ---------- ------- ---------- ------- ----------

함수 $f(x)$는 $x = 1$에서 극소이다.

따라서 $a = 1$

$f(a) = f(1) = 1^{3} - 3 \times 1 + 12 = 10$

이므로

$a + f(a) = 1 + f(1) = 1 + 10 = 11$
