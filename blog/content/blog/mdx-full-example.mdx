## 코드

인라인 코드는 `const x = 1` 처럼 백틱으로 감쌉니다.

코드 블록은 아래처럼 씁니다 (언어 지정 시 문법 강조 가능):

```js
// lib/image-url.ts와 동일한 논리: 베이스 URL과 경로 결합
function toImageUrl(src) {
  if (!src || src.startsWith("http")) return src;
  const path = src.startsWith("/") ? src : `/${src}`;
  return IMG_BASE_URL ? `${IMG_BASE_URL.replace(/\/$/, "")}${path}` : path;
}
```

## WhyItMatters (왜 중요한가)

"왜 중요한지"를 자유 문단으로. **자식(children)**으로 내용을 넣습니다.

<WhyItMatters>

작업기억은 **여러 단계**를 동시에 다뤄야 하는 과목에서 특히 중요합니다. 수학 불안이 있으면 불안 자체가 작업기억 자원을 쓰기 때문에, 실제로 풀어야 할 문제에 쓸 여유가 줄어듭니다. 그래서 부담을 줄이고 작은 성공을 쌓는 것이 중요합니다.

</WhyItMatters>
