# 작업기억(Working Memory)

## 정의

**작업기억**이란 동시에 머릿속에 유지하면서 조작하는 정보의 양을 말합니다. 수학에서는 식을 기억하면서 단계별로 계산하는 데 필수입니다.

## 핵심 요약

- 용량이 제한되어 있어, **인지부하**가 높으면 실수가 늘어납니다.
- 수학 불안이 있으면 불안 자체가 작업기억을 잠식해 성능이 떨어질 수 있습니다.

## 예시

- 두 자리 곱셈을 "한 번에" 하기보다 단계를 나누면 작업기억 부담이 줄어듭니다.
- 암산보다 종이에 쓰면 부담이 줄어듭니다.
